Portfolio

CloudFront Auth

An open source project I developed that allows users to authenticate before accessing data served via AWS CloudFront. Viewer requests are handled with Lambda@Edge to authorize and authenticate users. I have implemented support for Microsoft, Google, and GitHub login. After authorization/authentication, user login is persitent with the use of JWT's. An engineering article I wrote on this can be found on Widen's Engineering page. The project itself is available on GitHub


Metadata Extractor

A Java-based library that extracts metadata from numerous file formats. As a contributor to this open source project, I implemented support for QuickTime, MP4, HEIF, EPS, and more. Along with those implementations, I collaborated with other contributors to fix various bugs. Metadata Extractor is also available on GitHub. My contributions can be viewed here.


gALPHA by gener8or

I participated in a startup program that teaches participants how to create a successful startup. Developed a product to notify the user of approaching strangers while walking on campus. This idea came from my teammate who wanted to make students feel safer on campus following an attemptted abduction. Throughout the program, we aimed to develop an MVP and talk to businesses to help form our target audience.
